JS Digital Clock

Life and Times of a JavaScript Digital Clock, June 22nd, 2019

A simple little use of the Date() method.

'Twas a surprise to me how easy it was to make this JavaScript Clock, albeit a digital one. I must admit, it is much easier than an analog clock, CSS-wise. Easy peasy. All you do is:

  • Create your standard HTML page<!DOCTYPE html> and all. For ease just do your CSS inline with a <style> tag.
  • Add 1 <div> to the body. Give it an id and a class. I chose #MyClockDisplay and .clock.
  • Add a <script> tag and inside the body
  • Write a function, showTime() inside said function we need to set the, let's call them, 'parts of time' to variables.

function showTime(){

let date = new Date(); //The date constructor sets the date variable to a new Date object

let h = date.getHours(); //Sets h variable to whatever the current Hours is.

let m = date.getMinutes();//Sets m variable to whatever the current Minute is.

let s = date.getSeconds();//Same as above.

let session = "A.M.";//You will see below how this is used.


If you want 12 hour time

if(h == 0){

h = 12;

}

if(h > 12){

h -= 12

session = 'P.M.'

}

This is strictly for aesthetics

h = (h < 10) ? '0' + h : h;

I used the ternary operator here^. Basically this says: if the hour is less than 10. h is going to be '0' + h. (put a zero before the hour) if it's not less than 10, it's just h.

m = (m < 10) ? '0' + m : m;

s = (s < 10) ? '0' + s : s;

Add it all together into a time variable.

let time = h + ':' + m + ':' + s

Grab the #id to place this time variable in the html

document.getElementById('MyClockDisplay').textContent = time;


This setTimeout allows our function to be called, every second. (1000 milliseconds)

setTimeout(showTime,1000)

}

Call the function

showTime();

You can really see it coming together below.



All that is left is to style as you see fit. I centered my clock and used a text-shadow that I think is pretty BA, if you ask me.
